## 🌐 组织简介

**CPHOS** 由十余名**国家集训队**成员于 2020 年年末组建，经过 5 年的发展，现已有成员接近两百名。作为公益性组织，我们始终致力于：

* 🏗️ 为物理竞赛选手构筑透明开放的交流平台
* 🚀 为退役选手的全方位发展提供支持与帮助
* 🩺 为物理竞赛的健康蓬勃发展贡献力量

自成立以来，CPHOS 已组织公益性模拟考试 **27 次**，参与联考学校逾百所，单次参与人数稳定在千人规模。📢 **“CPHOS物理竞赛联考”** 微信公众号持续发布备考经验、试题简报、物竞快讯等高质量内容。

## ✉️ 理事长致辞

> “回首过往，这已是 CPHOS 走过的第五个年头。”

没有一劳永逸的方法，亦如没有十全十美的学说。我们怀抱着对物理竞赛发展的赤忱热情出发，为提高模拟卷质量而付出，迄今为止的联考就是最好的注脚。✨

随着成员们步入大学，我们不仅关注竞赛，更思考如何支持选手的未来。聚餐与沙龙孕育而生，不同方向的学长在此分享经验。新鲜血液的注入为我们带来了源源不断的活力。最初批卷用 PDF 翻阅、服务器自掏腰包维护的场景仍历历在目（笑），但感谢**衍复**的赞助，让我们能大步前行。

下一个五年，新加入的我们，正年轻的我们，会与 CPHOS 一直走下去。

**希望 CPHOS 永远年轻，永远正直，永远心怀理想，永远充满希望！**
